# VRFlite with IPSEc VTI

Another simple VRF setup with two seperate VRF instances. Two routers in instance 1 have a VTI connecting them. 

## Key Concepts
- Understand how a VRF process works on a router
- Understand how to configure and setup VRFs and apply them to interfaces 
-Understanding how route tables work with multiple VRF instances
-Configure and understand IKE phase1/phase2 setup and verification
