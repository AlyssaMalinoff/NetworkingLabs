# OSPF Stub Area Overview

Multi-Area OSPF lab with each type of Stub area (Stub, totally stub, NSSA, totally NSSA) and a regular area. This was a fun lab I made for some practice reading different route identifiers and general ospf configuration

## Key Concepts
- Understand how to enable OSPF on a router without using network statements under the process ID
- Understand how the global process and the per-interface process differ 
- Configure various stub area setups and links
- Examine route table changes in stub areas
